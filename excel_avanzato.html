    PowerQuery - importare diversi file excel in un unico foglio
    Tramite PowerQuery possiamo importare diversi fogli excel con le stesse caratteristiche, magari uniformandoli con delle funzioni custom, in un unico foglio excel per poi creare delle PivotTable o dei PivotChart. Da PowerQuery editor selezionare New Source - File – Folder – import all file come mostrato nella immagine sottostante :




    i fogli excel sono così stato caricati.





    Selezionare la colonna Content e premere il tasto dx del mouse e selezionare remove other columns.





    Ora selezionare nella barra dei Menu Add column e poi Custom column – inserire il commando Excel.Workbook([Content],true) e premere Ok.





    A questo punto abbiamo :





    Posizionarsi sul quadrato a dx nella colonna custum e dopo aver cliccato comparirà il riquadro seguente :





    Togliere la selezione a Use original custum name as prefix e premere Ok.





    Selezionare le colonne Name e Data poi Rimuovere le altre colonne :





    Espandere la colonna Data selezionando le colonne che si voglio visualizzare





    Sotto il risultato dove abbiamo i dati di tutti i fogli excel importati.





    Ora con i dati si può operare sia con PowerQuery per creare ad esempio delle correzioni e creare delle di aggrazione e poi riportarli in un classico foglio exec, fare grafici ecc..

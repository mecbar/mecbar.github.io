    • MULTIPLE LINEAR REGRESSION
    Con la regression possiamo creare un modello per effettuare le previsioni in base ai dati inseriti nella fase di training. Abbiamo 2 tipi di regression la Linear Regression con una sola variabile indipendente e la Multiple Linear Regression data dalla formula
    Y = a + bX1 + cX2 + nXn dove X1 .. Xn sono le variabili indipendenti.

    Di seguito viene presentato un esempio di un modello per predire i delitti in diverse cittadine degli USA.
    Avendo a disposizione molte variabili indipendenti prima con PCA individuiamo le 3 più significative e poi con la MULTIPLE LINEAR REGRESSION creiamo un modello per effettuare le previsioni.
